### Development setup

1. Clone the repo
2. `docker-compose build`
3. `docker-compose run --rm web bundle install`
4. `docker-compose up`
5. Your server awaits on port 3000

##### Potential issues

Make sure port 3000 is being forwarded from docker machine to your machine.
